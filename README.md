# Weather-App
Weather Android App for getting data about current and forecast weather for 7 days and 24 hours at a day.

<h1>Features.</h1>
<p><b>1. SDK Version with API 23+.</b></p>
<p><b>2. Uses Combination of BroadCastReceiver + AlarmManager on device with API 14 -22.</b></p>
<p><b>3. Use API for integrating service of forcast and current </b></p>
<p><b>4. Using Notifications for alert and the weather temperature when you use app and when you close it </b></p>
<p><b>5. Using Google Map to search about city and to favourite that city </b></p>
<p><b>6. Using System units to convert the temperature and wind </b></p>
<p><b>7. Using Language for change it </b></p>
<p><b>8. Using GPS in settings to know about your location weather</b></p>
